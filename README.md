## ⬡ Engineering Philosophy

<div align="center">

> *"I don't just write code that works — I write systems that hold."*

</div>

I'm a backend-focused engineer deeply invested in the **mechanics of distributed systems**, **real-time communication**, and **network security**. Currently pursuing a BS in Computer Science (6th Semester), I treat each project as a proving ground for production-grade thinking — obsessing over connection handling, fault tolerance, and protocol design, not just feature delivery.

I believe the best engineers understand their stack at every layer — from the TCP handshake to the application logic riding on top of it.
